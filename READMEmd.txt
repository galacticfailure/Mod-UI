## Overview
FJFE is meant to add quality-of-life features for both new and old mods. It's not an overhaul of FJ by any means, just some stuff to make the site easier to use and moderate.

# Installation
**1.** Determine what version you need. `FJFE-Main` is built for all (or at least most) browsers, `FJFE-Phaser` is built for Firefox-based browsers. Firefox itself won't let any unsigned (nonpublic) extensions be added. Some Firefox-based browsers (forks) let you do it, but not all, so make sure yours supports unsigned extensions before you proceed.
**2.** Navigate to the latest version and download the branch you need.
**3.** Proceed according to the branch you installed:
**- FJFE-Main:** Extract the .zip and place the folder somewhere you'll remember. Navigate to your browser's Extensions page and enable `Developer Mode` (usually in the top right). Press `Load Unpacked` (usually top left), then select the unzipped folder. FJFE should load into the extension page. Refresh FJ once finished. Note: If you move or delete the folder on your PC, it will make updating the extension slightly more difficult, though minimally so.
**- FJFE-Phaser:** Navigate to `about:config` in your URL, search for `xpinstall.signatures.required`, set it to `false`. Once complete, install the .xpi and enable the extension as needed. Refresh FJ once finished.
**4.** The extension will add a **Mod UI** button to the sidebar; if it's there, it's working correctly.
**Note:** the extension's main settings menu is in FJ's sidebar, above the NSFW button. If you have `Show extended sidebar on desktop` (FJ settings) disabled, it will not show up, so make sure it's enabled.

# Updating
The extension will display your local version; new versions will be posted when available, just compare the version numbers. Because this isn't a public extension, updates aren't automatic.
**- FJFE-Main:** Follow step 2 from Installation. Delete the original folder, replace it with the new (unzipped) folder. Navigate to your Extensions page and reload the extension, then refresh FJ and check for updates again to make sure it installed correctly.
**- FJFE-Phaser:** Navigate to your Extensions page and delete the extension, then follow the installation instructions to reinstall the new version.
**Note:** It should go without saying, but you don't need to download every FJFE version. Just whenever you feel like it (or, in the case of students, when your instructor recommends it).
